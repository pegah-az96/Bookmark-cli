### راهنمای استفاده از برنامه
>##### bookmark -h
>##### bookmark --help 

**مشاهده تمامی لینک های ذخیره شده**
>##### bookmark read

**ذخیره لینک جدید**
>##### bookmark insert -l "url" -d "your description" -t "your tags"

**حذف لینک مورد نظر**
>##### bookmark delete id

**جستجوی تگ و مشاهده لینک های  حاوی تگ** 
>#####  bookmark search tagname
